# Screenshots for the README

The main README references two images that make it land for a portfolio. Add them here with these exact filenames.

## 1. `docs/dashboard.png`  (the results dashboard)

This is the single most important image. It shows the finding at a glance.

- Open the results dashboard artifact (the one titled "Is a second agent worth paying for?").
  If you do not have the link handy, ask me to list your artifacts.
- Capture the top of the page: the three big stat tiles and the "forest plot" chart underneath.
- Save it here as `dashboard.png`. A wide screenshot (about 1200px) looks best.

## 2. `docs/app.png`  (the doodle web demo)

Shows the interactive side of the project.

- Run `python Gold/app.py`, open `http://127.0.0.1:5000/app`.
- Enter your key, then capture the page showing the title, the four mascot models, and one analyzed result
  (paste a sentence and press analyze first, so there is something on screen).
- Save it here as `app.png`.

## Optional: `docs/journey.png`

The two-act "journey" timeline artifact also makes a nice image if you want a third. It is not referenced in the
README by default; add an image tag for it if you use it.

## Tips

- On macOS, Shift+Cmd+4 lets you drag a rectangle to capture part of the screen.
- Crop out browser chrome (tabs, address bar) so only the page content shows.
- Keep the files reasonably small (under ~500 KB each) so the repo stays light.
