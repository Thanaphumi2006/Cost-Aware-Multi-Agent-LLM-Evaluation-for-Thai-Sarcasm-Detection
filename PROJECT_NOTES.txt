========================================================================
 THAI SARCASM DETECTION — PROJECT NOTES
========================================================================

Goal: build a human-labeled gold set for the "sarcastic (1) / not sarcastic (0)"
      classification task, then build a baseline to compare against more complex
      (multi-agent) systems later.

Labels:  1 = sarcasm/irony   |   0 = not sarcasm   |   X = undecidable / needs outside context


========================================================================
 PART 1 — WHAT EACH FILE DOES
========================================================================

--- scripts (code) ---

Load_data.py
    Load the Thai datasets from Hugging Face (Wongnai + Wisesight) into one table
    -> writes raw_texts.csv  (72,865 texts, columns text, source)

First_round_Test/Keyword Filter for NLP Project.py
    Keyword step: score "sarcasm suspicion" using keywords/patterns
    (praise word + bad context = strong signal)
    -> scored_texts.csv  (every item + suspect_score + signals)
    -> to_label.csv      (400 sampled = 240 high-suspicion + 160 normal, to label)

First_round_Test/Label/Lim_prelabel.py   [uses OpenAI GPT-4o | needs OPENAI_API_KEY]
    Have the LLM draft a label per item (llm_label, llm_reason) from to_label.csv
    Saves every 10 items, resumable
    -> to_label_prelabeled.csv
    ** these are only drafts; a human must review/fix them **

First_round_Test/Label/human_review.py   [no API key — interactive]
    Human reviews/fixes the draft labels one by one
    - MODE="blind": hides the LLM answer, you decide first, then reveal -> reports % agreement with the LLM
    - MODE="fast" : shows the LLM answer, Enter=agree, type=fix
    -> to_label_reviewed.csv (adds human_label) and builds gold.csv

First_round_Test/Label/baseline_zeroshot.py   [uses GPT-4o | not run yet]
    "Single agent" zero-shot LLM baseline: predicts every item in gold.csv
    Measures accuracy / precision / recall / F1 (positive class = sarcasm) + confusion matrix
    -> baseline_preds.csv  (keeps F1 as the baseline)

Gold/Check_gold.py   [no API key]
    Check whether gold.csv is measurable: count, 1/0 ratio, warnings about skew

Gold/adjudicate.py   [no API key — interactive | not run yet]
    Walk through only the items where human vs LLM disagree (27 items)
    Use alongside labeling_rubric.md -> update final_label -> rebuild gold.csv

Gold/Harvest.py   [uses GPT-4o | needs OPENAI_API_KEY]
    Mine more "sarcasm suspects" from the big scored_texts.csv (68k)
    Focus on Wisesight + short texts + sorted by high suspect_score, LLM filters
    -> Gold/harvest_to_review.csv (a sarcasm-dense pile, for further human review)

--- documents ---

Gold/labeling_rubric.md
    The labeling rubric (sharp version), separating the boundaries:
    sarcasm vs. direct criticism vs. balanced review vs. sincere praise, with real examples from our data

NLP Project Proposal README.md
    The project proposal document (original)

--- data files (results) ---

raw_texts.csv .............. 72,865 raw texts (Wongnai+Wisesight)
scored_texts.csv .......... every item + suspicion score (from keywords)
to_label_prelabeled.csv ... 400 items + GPT draft labels
to_label_reviewed.csv ..... 400 items + human_label (116 reviewed)
Gold/gold.csv ............. the gold set (1/0 labels only) = 101 items (1:7, 0:94)
Gold/harvest_to_review.csv  extra mined sarcasm suspects = 495 items (has a label column to review)
                            ** LLM over-flags: many, but many aren't really sarcasm; needs human review **
